Caenorhabditis sp. 26 is a gonochoristic species belonging to the Elegans supergroup. The sequenced strain, JU2190, is an inbred strain generated from the type isolate JU2161, which was isolated in Zanzibar, Tanzania (pers. comm). See below for more information.

Species and strain

A formal description of C. sp. 26 is currently underway.

CGC strain entry: JU2190

Raw data

The raw data can be downloaded from either ENA or NCBI (please note the raw data contain sequencing adapters and bacterial contamination - contact lewis.stevens@ed.ac.uk for “cleaned” reads):

https://www.ebi.ac.uk/ena/data/view/PRJEB12596

https://www.ncbi.nlm.nih.gov/bioproject/PRJEB12596

Assembly

The genome of C. sp. 26 was assembled by Lewis Stevens (Blaxter Lab) using short-read data.

See our general assembly pipeline for more info: CGP Pipeline

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [1].
