Caenorhabditis virilis (formerly sp. 13) is a gonochoristic species belonging to the Drosophilae supergroup. The sequenced strain, JU1968, is an inbred strain generated from the type isolate, JU1528, which was isolated in Orsay, France [1]. See below for more information.

Species and strain

C. virilis (sp. 13) was formally described in [1]. Discussion about morphological characters can be found in [2] (termed sp. 13 here). C. imperialis is the most closely related species [1].

Zoobank entry: 50737CD2-403A-46C3-AC3E-DCF2222B2DAA

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [3].
