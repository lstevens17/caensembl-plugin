Caenorhabditis sp. 32 is a gonochoristic species belonging to the Elegans supergroup [1]. The sequenced strain, JU2788, is an inbred strain generated from the type isolate SB454, which was isolated from the gut of a millipede, Archispirostreptus gigas, in Africa (pers. comm). See below for more information.

Species and strain

A formal description of C. sp. 32 is currently underway. C. afra is the most closely related species [1].

CGC strain entry: JU2788

Raw data

The raw data can be downloaded from either ENA or NCBI (please note the raw data contain sequencing adapters and bacterial contamination - contact lewis.stevens@ed.ac.uk for “cleaned” reads):

https://www.ebi.ac.uk/ena/data/view/PRJEB12601

https://www.ncbi.nlm.nih.gov/bioproject/PRJEB12601

Assembly

The genome of C. sp. 32 JU2788 was assembled by Lewis Stevens (Blaxter Lab) using short-read data.

See our general assembly pipeline for more info: CGP Pipeline

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [2].
