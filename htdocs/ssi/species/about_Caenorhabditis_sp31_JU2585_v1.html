Caenorhabditis sp. 31 is a gonochoristic species which likely belongs to the Drosophilae supergroup (note: phylogenies using whole genome data [1] have frequently placed C. sp. 31 as sister to the Elegans supergroup. Further work is required here). The sequenced strain, JU2585, is an inbred strain generated from the type isolate JU2469, which was isolated in Madre de Dios, Peru (pers. comm). See below for more information.

Species and strain

A formal description of C. sp. 31 is currently underway.

CGC strain entry: JU2585

Raw data

The raw data can be downloaded from either ENA or NCBI (please note the raw data contain sequencing adapters and bacterial contamination - contact lewis.stevens@ed.ac.uk for “cleaned” reads):

https://www.ebi.ac.uk/ena/data/view/PRJEB12600

Assembly

The genome of C. sp. 31 JU2585 was assembled by Lewis Stevens (Blaxter Lab) using short-read data.

See our general assembly pipeline for more info: CGP Pipeline

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [2].
