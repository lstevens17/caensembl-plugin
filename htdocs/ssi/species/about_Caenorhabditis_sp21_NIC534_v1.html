Caenorhabditis sp. 21 is a gonochoristic species, and the second ealiest branching species in the genus. The sequenced strain, NIC534, is an inbred strain generated from the type isolate NIC134, which was isolated in Madagascar (pers. comm). See below for more information.

Species and strain

A formal description of C. sp. 21 is currently underway.

CGC strain entry: NIC534

Raw data

The raw data can be downloaded from either ENA or NCBI (please note the raw data contain sequencing adapters and bacterial contamination - contact lewis.stevens@ed.ac.uk for “cleaned” reads):

https://www.ebi.ac.uk/ena/data/view/PRJEB12595

https://www.ncbi.nlm.nih.gov/bioproject/PRJEB12595

Assembly

The genome of C. sp. 21 NIC534 was assembled by Lewis Stevens (Blaxter Lab) using short-read data.

See our general assembly pipeline for more info: CGP Pipeline

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [1].
