Caenorhabditis kamaaina (sp. 15) is a gonochoristic species belonging to the Elegans supergroup [1]. The type isolate, QG122, was collected Kauai, Hawaii. See below for more information.

Species and strain

C. kamaaina was formally described in [1]. Discussion about morphological characters can be found in [2] (termed sp. 15 here).

Raw data

The raw data are not yet available.

Assembly

The genome of C. kamaaina QG2077 was assembled by Luke Noble (Rockman Lab) using a combination of short-read and mate-pair data.

Annotation

NOTE: This is a preliminary gene set. We may update the gene set soon.

We predicted genes using AUGUSTUS. We did not have RNA-seq data.
