Caenorhabditis doughertyi (formerly sp. 10) is a gonochoristic species belonging to the Elegans supergroup. The sequenced strain, JU1771, is an inbred strain generated from the type isolate JU1333, which was isolated in Kerala, India [2]. See below for more information and references.

Species and strain

C. doughertyi (sp. 10) was formally described in [1]. Discussion about morphological characters can be found in [2] (termed sp. 10 here). C. brenneri is the most closely related species [1].

Zoobank entry: 6D1A7C02-982E-472E-8619-F6746AB78B51

CGC strain entry: JU1771

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [3].
