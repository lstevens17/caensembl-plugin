WARNING. This assembly is low quality. Our raw data is poor quality and the genome is heterozygous. We plan to resequence.

Caenorhabditis sp. 39 is a gonochoristic species belonging to the Elegans supergroup [1]. The sequenced strain, NIC564, is the type isolate isolated from Morne Trois Pitons, Dominica. See below for more information.

Species and strain

A formal description of C. sp. 39 is currently underway.

CGC strain entry: NIC564

Raw data

The raw data can be downloaded from either ENA or NCBI (please note the raw data contain sequencing adapters and bacterial contamination - contact lewis.stevens@ed.ac.uk for “cleaned” reads):

https://www.ebi.ac.uk/ena/data/view/PRJEB12602

https://www.ncbi.nlm.nih.gov/bioproject/PRJEB12602

Assembly

The genome of C. sp. 39 NIC564 was assembled by Lewis Stevens (Blaxter Lab) using short-read data.

See our general assembly pipeline for more info: CGP Pipeline

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [2].
