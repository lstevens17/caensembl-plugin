Caenorhabditis monodelphis (formerly sp. 1) is a gonochoristic species and the most early diverging Caenorhabditis species [1]. The sequenced strain, JU1667, is an inbred strain generated from the type isolate SB341, which was isolated from the fruting body of a fungus, Ganoderma applanatum, in Berlin, Germany [1]. See below for more information.

Species and strain

C. monodelphis (sp. 1) was formally described in [1]. Discussion about morphological characters can be found in [1,2].

CGC strain entry: JU1667

Raw data

The raw data can be downloaded from either ENA or NCBI (please note the raw data contain sequencing adapters and bacterial contamination - contact lewis.stevens@ed.ac.uk for “cleaned” reads):

https://www.ebi.ac.uk/ena/data/view/PRJEB7905

Assembly

The genome of C. monodelphis JU1667 was assembled by Lewis Stevens (Blaxter Lab) using short-read data.

See our general assembly pipeline for more info: CGP Pipeline

Annotation

Genes were predicted using BRAKER which utilises evidence from raw RNA-seq data [3].
